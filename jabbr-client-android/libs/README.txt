The project requires the following files present in this directory:
	- signalr-client-sdk.jar
	- signalr-client-sdk-android.jar
	- gson-2.2.2.jar


Run getLibs.ps1 or getLibs.sh to download gson-2.2.2.jar

To add the ActionBar dependency, add the android-support-v7-appcompat project located in the android SDK, and add a reference of the appcompat project to this project. 
For more instructions to install the project see:
http://developer.android.com/tools/support-library/setup.html#libs-with-res
